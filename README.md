# 100 Days of Listen Labs

Internal image generator — pick a theme, drop in a quote + your name/title, pick a card theme, and export/copy a shareable PNG.

## Develop

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build

```bash
npm run build
npm start
```

## Deploy

Push to GitHub, then import the repo into Vercel. No env vars required.
